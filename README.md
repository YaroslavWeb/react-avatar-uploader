# React avatar uploader (test-task)

## Stack

- React
- Redux
- Saga
- SCSS
- TypeScript
- Axios


## How to install?
1. `npm install` (client)
2. `cd ./server`
3. `npm install` (server)

## How to use?
1. `npm run start` (client)
### Second terminal
2. `cd ./server`
3. `node server`

## Server PART
- Uploaded images are stored in the folder "public".
- Server loads only PNG, JPG and JPEG images.
- Max size image = 10MB.
